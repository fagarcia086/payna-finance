## Getting Started

### Prerequisites

- npm

### Installation

1. Clone the repo

   ```sh
   git clone
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. To start development server

   ```sh
   npm start
   ```

   Development server located at `localhost:8000`

### Build

```sh
npm run build
```

Build output directory: `/dist`
